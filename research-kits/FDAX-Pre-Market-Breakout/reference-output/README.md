# Reference Output

This folder contains the pre-built study output for the FDAX pre-market breakout research.

| File | Description |
|------|-------------|
| `index.html` | Full interactive HTML report. Open in any browser — no server required. Contains KPI cards, breakout distribution charts, bucketed tables, and a findings summary. |
| `FDAX_Pre_Market_Breakout_Stats_v2_0_20260516.md` | Markdown stats summary. Every number from the HTML report in plain text, suitable for cross-study comparison or note-taking. |

Both files cover 1,878 trading days (Dec 2018 – May 2026).
